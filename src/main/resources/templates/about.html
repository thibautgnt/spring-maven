À propos de cette application

Application développée à l'EFREI Paris

Projet de cours DevSecOps

Cette application a été développée dans le cadre du cours de DevSecOps à l'EFREI Paris.

Développeur

Thibaut Genet
github.com/thibautgnt

Technologies utilisées

  • Spring Boot 3.4.4
  • Spring Data JPA
  • Thymeleaf
  • H2 Database
  • Bootstrap 5
  • Lombok

Fonctionnalités

  • Gestion complète des tâches (CRUD)
  • Filtrage par statut (actives/terminées)
  • Système de priorité
  • API REST complète
  • Interface utilisateur responsive

Principes DevSecOps appliqués

  • Validation des entrées utilisateur
  • Gestion sécurisée des données
  • Architecture en couches
  • Séparation des préoccupations (SoC)
  • Bonnes pratiques de développement
